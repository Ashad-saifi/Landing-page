# Quick Start: Workspace Panels

This document guides you through using the interactive workspace controls in the Developer Console.

## Opening the Workspace
Click the **Workspace** button on the right side of the floating navigation header.

## 1. Left Panel: Layout Controller
Use this panel to customize the site's layout parameters dynamically using React state hooks:
- **Brand name**: Modifies logo titles.
- **Hero Title**: Directly alters the primary landing page header.
- **Theme Selection**: Choose between Violet, Emerald, Orange, or Indigo themes to adjust core highlights and hover colors.
- **Toggle Modules**: Hide or show individual feature blocks to test responsive layout spacing.

## 2. Center Panel: Live Viewport Simulator
Displays the customized web layout inside a mock browser container:
- **Device Mode**: Click Desktop, Tablet, or Mobile icons at the top toolbar to scale viewport boundaries instantly.
- **Templates**: Select SaaS, Agency, or Enterprise styles to swap core graphics.

## 3. Right Panel: Leads Monitor & Console Logs
- **Database Leads**: Displays enquiries submitted on the customer contact form. Clean resolved cards or delete obsolete database messages.
- **Console Output Logs**: Live system logger displaying actions performed in the workspace in real-time.
