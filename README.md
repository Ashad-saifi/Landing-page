# WebSolution Pro | Premium Web Application

A full-stack, responsive web application and developer customization workspace built with React, Vite, Tailwind CSS (v4), and Node.js/Express.

## Features
- **Modern Landing Page**: Stunning dark/light mode SaaS-inspired landing page with glowing gradient backdrops, floating layouts, and smooth animations.
- **Interactive Workspace Panel**: Multi-device viewport preview simulator (Desktop, Tablet, Mobile) with layout presets (SaaS, Agency, Enterprise).
- **Express Backend Server**: Dedicated Node/Express backend that receives inquiries, validates payloads, and saves messages to a JSON database file.
- **Admin Lead Manager**: Integrated leads console to monitor, mark resolved, or delete contact submissions.
- **Console Log Stream**: High-fidelity terminal log displaying runtime operations.

## Quick Start

### Installation
Install project packages:
```bash
npm install
```

### Run Locally
Launch the application:
```bash
npm run dev:all
```

- **Frontend Client**: [http://localhost:5173](http://localhost:5173)
- **Backend API**: [http://localhost:5000](http://localhost:5000)
